# cep
Teste para empresa nuclea.

Alguns padrões de projetos foi implementado como o adapter e arquitetura hexagonal.

Rota
```http
GET http://localhost:8080/cep/01001000
```

Realiza a instalação das dependências do projeto.
```bash
mvn clean install
```

Executa os testes.
```bash
mvn test
```

Executa o projeto.
```bash
mvn spring-boot:run
```
